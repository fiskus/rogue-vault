## Установка

`$ npm install`
`$ grunt`

### nginx conf

Add to `/etc/nginx/nginx.conf`:

    server {
        listen       1337;
        server_name  localhost;

        location / {
            root   /home/erokhin/documents/rogue-vault;
            index  index.html;
        }
    }
